# Dina Abdel Rahman Portfolio Website

This folder contains the full portfolio website files.

## Files included

1. `index.html`
   - This is the main website page.
   - It contains all website sections, text, buttons and structure.

2. `style.css`
   - This controls the design.
   - It controls colors, fonts, cards, spacing, mobile view and layout.

3. `script.js`
   - This controls the mobile menu.
   - It also updates the footer year automatically.

4. `Dina-Abdel-Rahman-Resume.pdf`
   - This is the resume file.
   - The "Download Resume" buttons point to this file.

5. `README.md`
   - This guide file.

## How to use on GitHub Pages

1. Create a GitHub account.
2. Create a public repository named:
   `dina-abdelrahman.github.io`

3. Upload all files in this folder to that repository.

4. Go to:
   Settings > Pages

5. Select:
   Source: Deploy from a branch
   Branch: main
   Folder: /root

6. Save.

Your website should become available at:
`https://dina-abdelrahman.github.io`

## What to change before publishing

In `index.html`, find this line:

```html
<a href="#" class="secondary-button">LinkedIn Profile</a>
```

Replace `#` with Dina's real LinkedIn profile URL.

Example:

```html
<a href="https://www.linkedin.com/in/your-linkedin-id/" class="secondary-button">LinkedIn Profile</a>
```
