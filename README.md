# Dina Abdel Rahman Portfolio

This fixed version uses inline CSS inside `index.html` so the website will not break if GitHub Pages does not load `style.css` correctly.

Upload these files to the root of the GitHub repository:

- index.html
- Dina-Abdel-Rahman-Resume.pdf
- style.css
- script.js
- README.md
- .gitignore

Then wait 1-3 minutes and refresh the website with Ctrl + F5.
