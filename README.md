# Dina Abdel Rahman Portfolio Website

This is a static portfolio website for Dina Abdel Rahman, Lead Data, AI and Cloud Architect.

## Files

- `index.html` - the full website content
- `style.css` - the visual design and responsive layout
- `script.js` - small mobile menu and footer year script
- `Dina-Abdel-Rahman-Resume.pdf` - latest resume PDF
- `.gitignore` - standard helper file

## How to update GitHub Pages

1. Open the GitHub repository: `dinaabdelrahman.github.io`
2. Click **Add file** > **Upload files**
3. Upload these files:
   - `index.html`
   - `style.css`
   - `script.js`
   - `Dina-Abdel-Rahman-Resume.pdf`
4. Click **Commit changes**
5. Wait 1 to 3 minutes
6. Refresh: `https://dinaabdelrahman.github.io/`

## Custom domain
